Thema dieses Fragebogens ist die oberflächliche Ersteinschätzung von Machine Learning (ML) Use Cases im Fernwärmenetzbetrieb. Durch den zunehmenden Digitalisierungsgrad und stetig steigende Datenverfügbarkeiten bieten sich immer mehr Chancen das Potential von ML Use Cases zu nutzen, um Aspekte wie Effizienz, Zuverlässigkeit und Nachhaltigkeit von Fernwärmenetzen zu verbessern. Die Identifizierung und Priorisierung potenzieller ML Use Cases stellt allerdings oft eine hohe Hürde dar, da einerseits unklar ist, was möglich ist und andererseits die Einschätzung aller Möglichkeiten komplex und zeitaufwändig ist. Um dieses Problem anzugehen, bietet Ihnen der ML4FW Fragebogen die Möglichkeit mit geringem Aufwand eine Übersicht über eine breite Auswahl an ML Use Cases im Fernwärmentzbetrieb zu erhalten, wobei erste Einschätzungen zu Nutzen, Aufwand und Risiko der Use Cases individuell auf Ihre Umstände und Präferenzen abgestimmt sind.
In diesem Fragebogen können 5 Kategorien von ML Use Cases im Fernwärmenetzbetrieb betrachtet werden (mehr dazu im späteren Verlauf des Fragebogens) und innerhalb der Kategorien stehen insgesamt 19 beispielhafte Use Cases zur Bewertung bereit.
Für eine sinnvolle Bewertung der Use Cases werden einige Fragen zum den Themen Datenverfügbarkeit, Datenqualität und Verfügbarkeit von Metadaten gestellt, da diese entscheidend für die Umsetzung von ML Use Cases sind. Für die Einschätzung des Nutzens und Aufwands der ML Use Cases wurden Kriterien definiert hinsichtlich derer Experteneinschätzungen getroffen wurden. Ziel dieses Fragebogens ist es dieses Wissen mit Ihren Präferenzen zu verbinden, um eine individuell auf Sie angepasste Einschätzung der Größen zu erhalten. Dafür sind an verschiedenen Stellen im Fragebogen Gewichtungen von Kriterien vorgesehen.
